# Portfolio

Personal portfolio site, a "Terminal CLI" themed build: monospace type,
green-on-black phosphor palette, bracketed buttons, and terminal-window panes
instead of cards.

Built with Next.js, TypeScript, and Tailwind CSS v4.

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Content

Site content (bio, projects, experience, skills) lives in
[`src/lib/data.ts`](src/lib/data.ts) — edit that file to update what's
displayed, no need to touch the page components.

## Build

```bash
npm run build
```
