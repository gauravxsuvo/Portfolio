Where these come from

Nothing here is a checked-in asset — every PNG is generated from code at build time, so the images can never drift from the site's palette:

src/lib/og-theme.ts — colours, tracks the default retro palette
src/lib/og-mark.tsx — the chevron + cursor mark
src/lib/og-crt.tsx — scanlines, glow, corner falloff
src/lib/og-banner.tsx — the home card
src/app/icon.tsx, src/app/apple-icon.tsx — icons
src/app/projects/[slug]/opengraph-image.tsx — project cards
